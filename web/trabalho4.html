Trabalho 4

Faces visíveis a partir de um ponto

Alunos: Hugo França
Professores: Afonso Paiva e Luis Gustavo Nonato

1. Descrição do algoritmo

Neste trabalho foi implementado um algoritmo para calcular as faces visíveis de um objeto convexo a partir de um ponto. O algoritmo possui as seguintes entradas e saídas:

Entradas:

  • Nuvem de pontos: o algoritmo recebe uma nuvem de pontos armazenadas em uma matriz \( V \).
  • Ponto: um ponto \( p \).
Saídas:
  • Faces vísiveis: o algoritmo retorna o conjunto de faces visíveis do fecho convexo de \( V \) a partir de \( p\).

De forma muito sucinta, o algoritmo funciona através dos seguintes passos:

  1. Calcular o fecho convexo \( Conv(V) \) de \( V \).
  2. Construir a corner table de \( Conv(V) \).
  3. Percorre cada face de \( Conv(V) \) até encontrar uma face \(F_1\) que é visível. Adiciona \( F_1 \) em uma pilha.
  4. Enquanto a pilha não estiver vazia:
    1. Retira uma face \( F\) da pilha.
    2. Se \(F\) já estiver sido detectada como visível anteriormente, ou se \(F\) não for visível, pula para a próxima iteração.
    3. Adiciona \(F\) na lista de faces visíveis.
    4. Insere as três faces vizinhas de \(F\) na pilha.

Para rodar o aplicativo implementado, basta rodar o arquivo main_app no matlab. Este aplicativo possui interface gráfica que permite modificar os parâmetros mais facilmente.
