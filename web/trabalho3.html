Trabalho 3

Geração de grids pelo método TTM

Alunos: Hugo França
Professores: Afonso Paiva e Luis Gustavo Nonato

1. Descrição

Neste trabalho foi implementado um algoritmo em Matlab para gerar grids através da resolução de equações elípticas pelo método TTM.

Os grid obtidos são comparados com outros obtidos pelos métodos TFI e Winslow. Além disto, realiza-se também um estudo de convergência de métodos iterativos utilizados na resolução das equações.

2. Resultados

Três geometrias foram testadas como exemplo. As parametrizações que definem cada uma destas geometrias podem ser vistas no início do arquivo de códifo-fonte build_tfi.m.

Em cada um dos exemplos foram realizados os seguintes testes:

  1. Cálculo de uma malha inicial \(M_1\) pelo método TFI.
  2. Cálculo de uma malha \(M_2\) pelas equações de Winslow (lado direito homogêneo).
  3. Cálculo de uma malha \(M_3\) pelas equações TTM (lado direito não-homogêneo).
Os testes 2 e 3 acima foram realizados 3 vezes cada um. Em cada uma das vezes, o sistema é resolvido por um método diferente, sendo eles: Jacobi, Gauss-Seidel e SOR.

Exemplo 1. Chevron

As figuras abaixo apresentam, respectivamente:
  • Malha gerada pela interpolação TFI;
  • Malha gerada pelas equações de Winslow resolvidas com método SOR;
  • Malha gerada pelas equações TTM resolvidas com método SOR. Os parâmetros usados foram: \(\xi_i = 0.5\), \(\eta_i = 0\) e \(a_i = b_i = c_i = d_i = 10\);
  • Convergência de cada um dos métodos iterativos para as equações TTM.

Observa-se que, no caso TTM as linhas são levadas mais a parte inferior-central do domínio. Isto acontece pois foram usados \(\xi_i = 0.5\) e \(\eta_i = 0\) .

Quanto a convergência dos métodos, observa-se claramente que o método SOR é o mais rápido, enquanto que o de Jacobi é o que mais demora a convergir.

Exemplo 2. Swan

Os mesmos testes realizados no exemplo Chevron foram feitos também para a geometria Swan. As imagens com os resultados equivalentes são apresentadas abaixo.

Nas equações TTM, os seguintes parâmetros foram escolhidos: \(\xi_i = 0.2\), \(\eta_i = 0.5\) e \(a_i = b_i = c_i = d_i = 10\). Observa-se que há uma concentração um pouco maior das linhas do grid próximas de \(x = 0.2\).

Exemplo 3. Airfoil

Os mesmos testes realizados nos exemplos Chevron e Swan foram feitos também para a geometria Airfoil. As imagens com os resultados equivalentes são apresentadas abaixo.

Nas equações TTM, os seguintes parâmetros foram escolhidos: \(\xi_i = 0.5\), \(\eta_i = 0.99\) e \(a_i = b_i = c_i = d_i = 10\). Devido aos parâmetros escolhidos, a concentração de linhas é maior no lado esquerdo da geometria, ao longo do eixo horizontal.
